# 02-Python分支循环

## if 语句

在Python中，要构造分支结构可以使用`if`、`elif`和`else`关键字，同时需要注意Python中没有用花括号来构造代码块而是**使用了缩进的方式来表示代码的层次结构**。此外，**连续的代码如果又保持了相同的缩进那么它们属于同一个代码块**，相当于是一个执行的整体。**缩进**可以使用任意数量的空格，但**通常使用4个空格**，建议大家**不要使用制表键**或者**设置你的代码编辑工具自动将制表键变成4个空格**。

使用`if...elif...else...`结构或者嵌套的`if...else...`结构。

## for in  循环

如果明确的知道循环执行的次数或者要对一个容器进行迭代（后面会讲到），那么我们推荐使用`for-in`循环。比如，用for循环实现1~100求和

```
sum = 0
for x in range(101):
    sum += x
print(sum)
```

`range`的使用参照下面的：

- `range(101)`：可以用来产生0到100范围的整数，需要注意的是取不到101。
- `range(1, 101)`：可以用来产生1到100范围的整数，相当于前面是闭区间后面是开区间。
- `range(1, 101, 2)`：可以用来产生1到100的奇数，其中2是步长，即每次数值递增的值。
- `range(100, 0, -2)`：可以用来产生100到1的偶数，其中-2是步长，即每次数字递减的值。

## while 循环

如果要构造不知道具体循环次数的循环结构，我们推荐使用`while`循环。`while`循环通过一个能够产生或转换出`bool`值的表达式来控制循环，表达式的值为`True`则继续循环；表达式的值为`False`则结束循环。
